## **How to use**

```bash
# Clone this repository
$ git clone https://github.com/johanmeci/frontend-layout.git

# Go into the repository
$ cd frontend-layout

# Install dependencies
$ npm install

# Run the app
$ npm start

# Run with gulp
$ gulp start
```
